## Description

App for learning English with flashcards and make notes. App with Authentication using Firebase. All data (words and notes) are saved using Firestore.

## Built With
React JS <br />
React Router <br />
Redux <br />
Redux Thunk <br />
date-fns <br />
Formik & Yup <br />
Styled Components <br />
Firebase & Firestore <br />
